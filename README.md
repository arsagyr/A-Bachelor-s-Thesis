# Веб-приложение

Программный код веб-приложения выпускной квалификационной работы по анализу и прогнозированию экономических показателей разных стран.

## Установка и запуск
```
# Создайте .env файл с вашими настройками
cp .env.example .env

# Отредактируйте .env под вашу конфигурацию
nano .env

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
python app.py
```
## Работа с приложением

Осуществляетя переход по ссылке, указаном env-файле и внесите данные загрузив csv-таблицу.
На веб-странице предоставляется нужный инструментарий для анализа данных.

## Импорт данных из CSV/Excel

### Формат CSV файла

Система автоматически определяет колонки по ключевым словам:

- **Страна**: country, страна, country_name, nation
- **Год**: year, год, date, период  
- **Экспорт**: export, экспорт, exports
- **Импорт**: import, импорт, imports
- **ВВП**: gdp, ввп, gross domestic product
